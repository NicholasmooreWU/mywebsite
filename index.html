Key SQL Implementation 
/* Optimized volatility calculation using Window Functions */
WITH stock_changes AS (
  SELECT 
    date, ticker, close,
    /* LAG() accesses previous row data without a self-join for O(n) efficiency */
    LAG(close) OVER (PARTITION BY ticker ORDER BY date) AS prev_close,
    LAG(searches) OVER (PARTITION BY ticker ORDER BY date) AS prev_searches
  FROM stock_data
)
SELECT 
  ticker, 
  AVG(close - prev_close) AS avg_daily_volatility
FROM stock_changes
WHERE prev_searches > 50
GROUP BY ticker
ORDER BY avg_daily_volatility DESC;

Simple SCM (Rust Implementation)

The Challenge

Building a minimal, content-addressed Version Control System (VCS) from scratch in Rust. The goal was to replicate core Git functionality—including repository initialization, staging, and commit history—using only standard libraries and cryptographic hashing for data integrity.

The Solution

Architected scm, a Rust binary that manages file snapshots using a Merkle-like tree structure. Implemented SHA-256 hashing to generate unique IDs for every file blob and commit object, ensuring data integrity. Designed the system to serialize metadata into JSON and store objects on disk, enabling features like `commit`, `log`, and `revert` (rolling back to previous states).

Key Systems Implementation 
use serde::{Deserialize, Serialize};
use sha2::{Digest, Sha256};
use std::collections::BTreeMap;

/// Content-addressable commit structure
#[derive(Serialize, Deserialize, Debug)]
struct Commit {
    tree: BTreeMap<String, String>, // Maps filenames to SHA-256 hashes
    parent: Option<String>,         // Parent commit hash
    message: String,
    timestamp: DateTime<Utc>,
}

fn cmd_commit(message: &str) -> Result<()> {
    let repo = repo_root()?;
    let index = load_index(&repo)?;
    
    let mut tree = BTreeMap::new();
    for rel in &index {
        let path = repo.join(rel);
        let mut f = fs::File::open(&path)?;
        let mut bytes = vec![];
        f.read_to_end(&mut bytes)?;
        
        // Store blob via SHA-256 hash
        let blob_hash = store_blob(&repo, &bytes)?; 
        tree.insert(rel.clone(), blob_hash);
    }

    let commit = Commit {
        tree,
        parent: read_head(&repo)?,
        message: message.to_string(),
        timestamp: Utc::now(),
    };
    
    // Serialize commit and update HEAD
    let commit_hash = commit_object_hash(&commit)?;
    write_head(&repo, &commit_hash)?;
    println!("Committed: {}", commit_hash);
    Ok(())
}
